### 结果分析

- **HWT（人类写作）**：精确率高（0.93）但召回率低（0.56），模型倾向于将人类写作误分类为模型生成文本
- **LGT（模型生成）**：召回率高（0.93）但精确率较低（0.74），模型对模型生成文本的识别能力较强
- **HLT（模型增强）**：整体表现最好，F1 达 0.87
